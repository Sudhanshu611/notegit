# System Design Fundamentals

## Load Balancing Strategies

Load balancing is a critical component of highly available system designs. It distributes incoming network traffic across multiple servers to ensure no single server becomes a bottleneck.

  • - Round Robin: Sequential distribution.
  • - Least Connections: Sends requests to the server with fewest active sessions.
  • - IP Hash: Uses client IP to determine server assignment.
