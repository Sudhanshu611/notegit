Implementing Balanced BST Nodes

When implementing a self-balancing tree, the rotation logic is critical. The height of each node must be updated after every insertion and deletion to ensure the O(log n) property holds.

function leftRotate(x) {
    let y = x.right;
    let T2 = y.left;
    y.left = x;
    x.right = T2;
    x.height = max(height(x.left), height(x.right)) + 1;
    y.height = max(height(y.left), height(y.right)) + 1;
    return y;
}
